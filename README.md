# presentations

公開用のプレゼン資料・配布資料を管理するリポジトリです。

## 公開ページ
- GitHub Pages: https://kiyochika.github.io/presentations/

## 置いているもの
- `docs/` : GitHub Pages で公開するファイル
- `docs/slides/` : 公開用 PDF
- Releases : PowerPoint 原本や配布用ファイル

## 運用方針
- ブラウザ閲覧用は PDF を `docs/slides/` に置く
- PowerPoint 原本（`.pptx`）は Releases に置く
- `index.html` から各資料にリンクする

## メモ
通常の GitHub リポジトリでは大きいファイルに制限があるため、
容量が大きい配布ファイルは Releases を使う想定です。
